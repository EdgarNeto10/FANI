Instruções

1- Para inserir dados a uma nova passagem ou passagem filha deve preencher o formulário à direita “Escrever narrativa ”

2- Para adicionar uma nova passagem de clicar em criar uma nova passagem.

3- Para adicionar uma passagem filha deve clicar primeiro no “node” que pretende expandir e de seguida clicar em passagem filha.

4- Para guardar a narrativa deve clicar em salvar a narrativa e adicionar um nome para mesma.
